						GitHub-Guide
|===================================|
|how update changes to github server|
|===================================|
git add file.*
git commit -m "some message on changes"
git remote add origin https://path/git/repository -> create remote directory
git push origin master -> direct changes to origin/master

		
Switch between GitBranches
git checkout -b branch_name ->create new branch
git checkout branch_name -> switch to another created branch

Merging branch to origin/master
git push --set-upstream origin branch_name -> to merge the branch with origin/master


|====================|
|how to configure git|
|====================|
sudo apt install git
git init directory_name_on_local_computer ->create local git repository
git config --global user.email "email.id" ->create git user_email
git config --global user.name "username"  ->create git user_name
